WELCOME!

Westfield’s API allows programmatic access to API endpoints that present related resources which collectively represent Westfield’s data.
Directory Services is the collection of endpoints that describe Centres and all related resources. Centres exist in multiple countries, and have API endpoints for information about hours, location, parking, events, deals/offers, movies, stores, products, product categories.
User Authentication APIs are a work in progress, and currently unavailable.
Platform Services can be used to complement the Westfield API assets.
