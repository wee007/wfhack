WELCOME!

The Westfield API allows a developer to interact with Westfield's data and services.
The Directory Content section contains APIs that allows you to work with centre information, such as opening hours, location, stores, contact details. It also enables access to centre-related information: events, deals and offers, movies.
Users and Authentication APIs are not ready for general use just yet.
Platform services and Micro services can be used to complement the Westfield API assets.
